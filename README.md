# AI Chatbot

Full-stack AI chatbot built with:

- React + Vite
- Django REST Framework
- JWT Authentication
- Gemini AI
- PostgreSQL (Supabase)

## Features

- User Registration/Login
- JWT Authentication
- Conversation History
- Auto Conversation Titles
- Rename/Delete Chats
- Markdown Rendering
- Token Refresh
- Responsive UI

## Tech Stack

Frontend:
- React
- Axios
- Tailwind CSS

Backend:
- Django
- DRF
- SimpleJWT

Database:
- PostgreSQL (Supabase)

AI:
- Gemini API
